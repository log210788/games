# ESL Memory and Connect Four Games 🎮

Welcome to the ESL Game Portal! This repository contains a collection of interactive, premium educational games designed for English learners to practice speaking, grammar, and vocabulary in classroom and online settings.

## 🚀 Play Online
The games are served live on GitHub Pages:
👉 **[log210788.github.io/games/](https://log210788.github.io/games/)**

## 🎮 Included Games

### Connect Four Series 🔴🟡
*   **Connect 4 Online (Real-time)**: Play live against students with automatic turn-syncing.
*   **Animals Edition**: Identify animals (Easy) or answer trivia (Hard) to drop tokens.
*   **Jobs & Careers**: Review occupation vocabulary and speaking prompts.
*   **Let's Chat!**: Answer conversation-starter questions to claim slots.
*   **Daily Habits**: Learn routines and healthy lifestyle terms.
*   **Frequencies**: Practice adverbs of frequency (always, usually, often, never).
*   **Simple Questions**: Review grammar and sentence structures.
*   **Sonic Edition**: Fast-paced arcade layout for younger learners.
*   **All Topics**: A mega-board covering Jobs, Animals, Food, Transport, Sports, and Fruits.
*   **Vietnamese ESL**: Bilingual hints and translations for local classes.
*   **Past Continuous**: Practice tenses with "was/were + verb-ing".
*   **How Often**: Answer adverb of frequency speaking prompts.

### Memory Match Series 🧠
*   **Brain Match!**: Multiplayer matching memory game with customizable grids.
*   **Clothes Memory**: Match clothing items with illustrations and Thai translations.
*   **Places Memory**: Match community locations and maps.
*   **Weather Memory**: Match animated weather GIFs and temperatures.
*   **Memory Tray Online**: Synchronized classroom board where teachers control the tray and students submit guesses.
*   **Jobs Memory**: Specialized career roles and helpers matching game.

## 🛠️ Local Development
To run these games locally, simply clone the repository and open `index.html` in any web browser:
```bash
git clone https://github.com/log210788/games.git
cd games
# Open index.html in browser
```
